\# Rewired

1.4.X versions of Baldi's Basics Classic use a third-party plugin called Rewired to handle their input, which is paid for and not open source.

This has been removed from any and all 1.4.X decompilations present in this repository, which means they may have slightly inaccurate input, and lack control rebinding, controller support, rumble, and mobile controls.

This is the only major change made to these decompilations.
